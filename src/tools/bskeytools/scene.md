# 场景管理工具

提供场景备份、图层管理、对象管理等实用工具。

---

## 💾 快速备份 (BsQuickSave)

自动备份系统，保护你的工作成果。

### 功能特点

| 功能 | 说明 |
|:-----|:-----|
| 自动备份 | 定时自动保存备份 |
| 版本管理 | 保留多个历史版本 |
| 快捷保存 | 一键快速备份 |
| 恢复功能 | 从备份快速恢复 |

### 设置选项

| 选项 | 说明 |
|:-----|:-----|
| 备份间隔 | 自动备份的时间间隔 |
| 保留数量 | 最多保留的备份数量 |
| 备份路径 | 备份文件存储位置 |

### 使用方法

**快速备份：**
- 点击主面板的 `快备份` 按钮
- 或使用快捷键（可自定义）

**自动备份：**
1. 打开备份设置
2. 启用自动备份
3. 设置备份间隔
4. 正常工作即可

---

## ⏰ 时光机 (BsOpenTools)

快速打开和收藏文件的工具。

### 功能

| 功能 | 说明 |
|:-----|:-----|
| 最近文件 | 显示最近打开的文件列表 |
| 收藏夹 | 收藏常用文件和文件夹 |
| 快速打开 | 一键打开文件 |
| 文件预览 | 预览场景缩略图 |

### 使用步骤

1. 点击 `时光机` 按钮
2. 在最近文件列表选择
3. 或从收藏夹快速访问
4. 双击打开文件

---

## 📂 图层管理 (BsLayerManager)

增强的图层管理工具，比原生图层管理器更强大。

### 功能特点

- 嵌套图层支持
- 批量图层操作
- 图层颜色管理
- 快速显示/隐藏

### 嵌套图层管理

支持嵌套结构的高级图层管理：

- 创建子图层
- 拖拽调整层级
- 批量移动对象到图层

### 使用步骤

1. 打开图层管理器
2. 右键创建新图层
3. 拖拽对象到图层
4. 使用快捷按钮控制显示/隐藏

---

## 🎯 选择集工具 (BsSelSetTools)

创建和管理选择集，快速选择常用对象组合。

### 功能

| 功能 | 说明 |
|:-----|:-----|
| 创建选择集 | 将当前选择保存为选择集 |
| 快速选择 | 一键选择整个选择集 |
| 添加/移除 | 编辑选择集内容 |
| 选择集管理 | 重命名、删除选择集 |

### 使用步骤

**创建选择集：**
1. 选择要保存的对象
2. 输入选择集名称
3. 点击创建

**使用选择集：**
- 点击选择集名称快速选择
- 按住 Ctrl 点击追加选择

---

## 🏷️ 对象批量重命名

强大的批量重命名工具，支持多种命名规则。

### 重命名模式

| 模式 | 说明 | 示例 |
|:-----|:-----|:-----|
| 前缀 | 添加前缀 | `Bone_` + Name |
| 后缀 | 添加后缀 | Name + `_L` |
| 替换 | 查找替换 | Left → Right |
| 序号 | 添加编号 | Name_001, Name_002 |

### 使用步骤

1. 选择要重命名的对象
2. 选择重命名模式
3. 输入重命名规则
4. 预览结果
5. 执行重命名

---

## 🎨 对象颜色工具

### 专业颜色工具

批量修改对象线框颜色，方便场景管理。

**预设颜色：**
- 按类型着色（骨骼、模型、辅助对象）
- 自定义颜色方案
- 随机着色

### 同色选择

选择场景中所有相同线框颜色的对象。

**使用步骤：**
1. 选择一个对象
2. 点击 `同色选择`
3. 自动选中所有同颜色对象

### 调色工具

调节选中对象的线框颜色。

---

## 🔍 对象选择器

高级对象选择工具，支持多种筛选条件。

### 筛选条件

| 条件 | 说明 |
|:-----|:-----|
| 名称 | 按名称筛选 |
| 类型 | 按对象类型筛选 |
| 颜色 | 按线框颜色筛选 |
| 图层 | 按所属图层筛选 |
| 材质 | 按材质筛选 |

### 创建带背景的选择面板

可视化的对象选择器，支持自定义背景图：

1. 创建选择面板
2. 设置背景图（如角色正面图）
3. 在背景上添加选择按钮
4. 点击按钮快速选择对应部位

---

## 🧹 场景清理 (CleanScene)

清理场景垃圾数据，优化文件大小。

### 清理内容

| 项目 | 说明 |
|:-----|:-----|
| 空组 | 删除空的组节点 |
| 无效材质 | 清理未使用的材质 |
| 无效图层 | 删除空图层 |
| 垃圾数据 | 清理临时数据 |

### 使用步骤

1. 打开场景清理工具
2. 选择要清理的项目
3. 预览清理内容
4. 执行清理
5. 保存场景

---

## 📊 场景信息 (PrintSceneInfo)

打印场景统计信息。

### 统计内容

- 对象总数
- 面数统计
- 材质数量
- 骨骼数量
- 动画帧范围

## 📥 批量导入导出

### 批量降版本

批量将 Max 文件降低版本保存。

**使用场景：**
- 向旧版本 3ds Max 用户分享文件
- 批量处理项目文件

**使用步骤：**
1. 选择源文件夹
2. 选择输出文件夹
3. 选择目标版本
4. 执行批量转换

### 批量导入转换

批量导入文件并转换格式。

**支持格式：**
- FBX
- OBJ
- 3DS
- MAX

---

## 🛡️ 病毒防护 (BsCleanVirus)

内置的 3ds Max 脚本病毒扫描和清理工具。

### 功能

| 功能 | 说明 |
|:-----|:-----|
| 启动扫描 | 打开文件时自动扫描 |
| 手动扫描 | 手动触发扫描 |
| 病毒清理 | 发现病毒自动清理 |
| 病毒库更新 | 更新病毒特征库 |

### 已知病毒

- ALC 病毒系列
- CRP 脚本病毒
- PhysX 病毒
- 其他已知变种

::: warning 建议
保持病毒库更新，定期扫描场景文件
:::
