视口显示工具

Bullet.S2026/1/2BsKeyTools视口显示预览大约 4 分钟

视口与显示工具

提供视口控制、显示模式切换、预览渲染等工具。

👁️ 显示控制

秀剪影

切换剪影显示模式，方便观察角色轮廓和姿态。

使用场景：

  • 审阅动画姿态剪影
  • 检查动作可读性
  • 参考电影分镜构图

使用方法：
点击 秀剪影 按钮切换开关

BOX 显示

将骨骼切换为 Box 显示模式。

优点：

  • 减少视口干扰
  • 提高性能
  • 更清晰的骨骼结构

冻结/解冻模型

一键冻结或解冻场景中的模型。

使用场景：

  • 防止误选模型
  • 专注于骨骼动画操作

显示/隐藏骨骼

功能说明
隐 Bone显示/隐藏 Bone 类型骨骼
隐 Biped显示/隐藏 Biped 骨骼
隐藏 Twist隐藏 Biped 的 Twist 骨骼

单模型预览

单独显示模型，隐藏其他所有对象进行预览。

使用步骤：

  1. 选择要预览的模型
  2. 点击 单模型 按钮
  3. 再次点击恢复显示

👻 洋葱皮 (Ghosting)

运动重影显示，查看动画前后帧的姿态。

功能

  • 显示前后 N 帧的重影
  • 可调整重影透明度
  • 可设置重影颜色

使用步骤

  1. 选中要显示重影的对象
  2. 点击 洋葱皮 按钮
  3. 设置重影帧数
  4. 播放动画查看效果

🔲 构图框

各种经典构图分割辅助线。

预设构图

构图说明
三分法经典九宫格构图
黄金分割黄金比例构图线
对角线对角线构图辅助
中心点中心交叉线
安全框标题/动作安全区域

使用方法

  1. 点击 构图框 按钮
  2. 选择构图类型
  3. 构图线显示在视口中
  4. 再次点击关闭

📹 视口工具 (BsVportTools)

视口管理和控制工具集。

相机视角管理

功能说明
保存视角保存当前视口视角
加载视角快速切换到保存的视角
视角列表管理多个保存的视角

视口配置

  • 一键保存/恢复视口布局
  • 快捷视图切换（前/后/左/右/上/下）

透视锁定

旋转视图时自动切换透视模式，避免在正交视图中意外旋转。

🖼️ 视窗切镜

储存和切换视角参数，用于多机位预览。

功能

  • 保存多个摄像机视角
  • 一键切换视角
  • 预览不同角度效果

使用步骤

  1. 调整到需要的视角
  2. 点击保存按钮
  3. 输入视角名称
  4. 需要时点击对应按钮切换

🎬 轨迹线

显示物体的运动轨迹线。

功能

  • 显示运动路径
  • 可视化速度变化
  • 显示关键帧位置

使用步骤

  1. 选中对象
  2. 点击 轨迹线 按钮
  3. 轨迹线显示在视口中

🎥 参考工具 (AnimRef)

在视口中播放参考视频或图片序列。

详细文档： 独立插件 - AnimRef

快速使用

  1. 点击 参考工具 按钮
  2. 导入参考视频/图片序列
  3. 设置同步选项
  4. 播放动画时同步显示参考

📷 快速拍屏

快速预览拍屏工具。

功能

  • 快速输出视口预览
  • 支持多种格式
  • 可设置输出分辨率

使用步骤

  1. 调整好视口视角
  2. 设置输出参数
  3. 点击拍屏
  4. 选择保存位置

🎨 作品预设

渲染作品的预设管理。

功能

  • 保存渲染设置预设
  • 快速切换不同渲染配置
  • 适用于不同输出需求

🔄 多向渲染

从多个角度批量渲染场景。

功能

  • 设置多个渲染角度
  • 自动旋转并渲染
  • 批量输出图片

预设角度

角度说明
8 向每 45° 一个角度
4 向前后左右
自定义自行设置角度列表

使用步骤

  1. 选择角色
  2. 选择渲染角度预设
  3. 设置输出路径
  4. 执行批量渲染

⏯️ 整小数帧

平滑拖动查看动画卡顿问题。

功能：

  • 在整数帧间平滑插值
  • 用于检测动画卡顿
  • 预览中间帧效果
