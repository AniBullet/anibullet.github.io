## ✨ 核心功能

### 🎬 动画制作

- **关键帧操作**：选择、移动、复制、粘贴关键帧
- **补间动画**：TweenMachine 快速创建补间
- **动画镜像**：左右对称动画复制
- **轨迹编辑**：可视化调整运动路径
- **动画烘焙**：约束转关键帧

### 🦴 骨骼绑定

- **Biped 工具**：专业 Biped 工具集
- **链条绑定**：样条线骨骼链系统
- **飘带解算**：简易动力学模拟
- **扭曲骨骼**：改善关节变形

### 🎨 蒙皮权重

- **权重编辑**：专业权重绘制工具
- **权重平滑**：分区平滑权重
- **蒙皮换骨**：替换骨骼引用
- **蒙皮拆合**：保留权重的拆分合并

### 📁 场景管理

- **自动备份**：保护工作成果
- **图层管理**：增强图层操作
- **病毒防护**：内置病毒扫描清理
- **批量处理**：批量导入导出

### 👁️ 视口工具

- **洋葱皮**：运动重影显示
- **构图框**：经典构图辅助
- **参考播放**：视口内播放参考视频
- **残影效果**：GhostTrails 插件
