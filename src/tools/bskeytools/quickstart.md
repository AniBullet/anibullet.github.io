# 快速入门

5 分钟快速上手 BsKeyTools。

---

## 📥 下载安装

### 自动安装（推荐）

1. 前往 [Releases](https://github.com/AniBullet/BsKeyTools/releases/latest) 下载最新安装包
2. 运行 `_BsKeyTools.exe`
3. 选择 3ds Max 版本，点击安装

![自动安装](https://raw.githubusercontent.com/AniBullet/BsKeyTools/main/SetupPrev.gif)

::: tip
安装完成后重启 3ds Max 即可使用
:::

---

## 🚀 启动工具

### 方式一：菜单栏

`菜单栏` → `BsKeyTools` → `BulletKeyTools`

### 方式二：工具栏

右键工具栏空白处 → 勾选 `BsKeyTools`

---

## 🎛️ 界面概览

工具界面分为三个主要区域：

| 区域 | 说明 |
|:-----|:-----|
| **顶部工具栏** | 关键帧操作、显示控制 |
| **快捷按钮区** | 常用功能一键访问 |
| **扩展面板** | 4 页签 × 12 按钮，可自定义排序 |

---

## ⚡ 常用功能速查

### 关键帧操作

| 功能 | 说明 |
|:-----|:-----|
| 选择帧前/后 | 快速选择当前帧前/后的关键帧 |
| 移动帧 | 批量移动选中的关键帧 |
| 选定帧范围 | 预览选择帧范围 |

### 显示控制

| 功能 | 说明 |
|:-----|:-----|
| 秀剪影 | 切换剪影显示模式 |
| 冻模型 | 一键冻结/解冻模型 |
| 隐 Biped | 显示/隐藏 Biped 骨骼 |
| 洋葱皮 | 运动重影显示 |

### 快捷工具

| 功能 | 说明 |
|:-----|:-----|
| 快备份 | 快速备份当前场景 |
| 暴力贴 | 动画复制粘贴 |
| 动画库 | Pose 库 / 参考库 |
| 时光机 | 快速打开历史文件 |
