骨骼绑定工具

提供 Biped、Bone 骨骼系统的专业绑定工具，包含链条绑定、扭曲骨骼、弹簧控制器等。

🦴 骨骼选择工具

选择所有 Biped

一键选中场景中所有 Biped 骨骼。

选择所有 Bone

一键选中场景中所有 Bone 类型骨骼。

选择所有骨骼

选择场景中所有骨骼对象，包括 Bone、Biped 和 CAT 骨骼。

🔧 Biped 专用工具 (BsBipedTools)

针对 Biped 系统的专业工具集。

功能列表

功能说明
快速选择选择特定部位的 Biped 骨骼
模式切换Figure/Motion/Footstep 模式快速切换
图层管理Biped 动画图层的快速操作
复制粘贴Biped 姿态的复制粘贴
镜像姿态左右对称镜像当前姿态

CS 选择工具

快速选择 Character Studio 骨骼的各个部位。

选择区域：

  • 全身 / 躯干 / 头部
  • 左臂 / 右臂 / 双臂
  • 左腿 / 右腿 / 双腿
  • 手指 / 脚趾
  • 尾巴 / 额外骨骼

🔄 TCB / 欧拉转换

在 TCB 和欧拉角控制器之间互相转换。

使用场景：

  • 导入 FBX 动画后的控制器调整
  • 修复旋转异常问题

使用步骤：

  1. 选择目标骨骼
  2. 选择转换方向（TCB→欧拉 或 欧拉→TCB）
  3. 执行转换

🌀 欧拉过滤

对 Bone 骨骼应用欧拉过滤，修复旋转翻转问题。

使用场景：

  • 修复 Gimbal Lock（万向锁）问题
  • 平滑旋转曲线

➕ 增删末端骨骼

为 Bone 骨骼链添加或删除末端骨骼。

功能：

  • 添加末端骨骼（用于蒙皮权重过渡）
  • 批量删除末端骨骼

🌀 Twist 扭曲骨骼

Twist 创建

创建扭曲骨骼系统，代替 Biped 自带的 Twist 骨骼。

优势：

  • 更灵活的扭曲控制
  • 更好的蒙皮变形效果
  • 支持自定义扭曲数量

使用步骤：

  1. 选择要添加扭曲的骨骼
  2. 设置扭曲骨骼数量
  3. 设置扭曲影响范围
  4. 创建扭曲骨骼

隐藏 Twist

快速隐藏/显示 Biped 的 Twist 骨骼。

🔗 链条工具 (Chain Tools)

使用样条线创建骨骼链绑定，适用于尾巴、脊椎、触手等柔性结构。

详细文档： 链条工具详解

快速使用

  1. 创建一条样条线作为路径
  2. 打开链条工具
  3. 设置骨骼数量和属性
  4. 选择控制方式
  5. 点击创建

🌊 飘带解算

飘带和布料的简易动力学解算工具。

基础版

功能：

  • 简单的跟随动力学
  • 适用于飘带、头发等

使用步骤：

  1. 选择要添加动力学的骨骼链
  2. 设置弹性和阻尼参数
  3. 播放动画查看效果
  4. 烘焙动画

增强版 (SpringMagic)

额外功能：

  • 更多参数调节
  • 碰撞检测
  • 风力影响
  • 批量处理

参数说明：

参数说明
Stiffness刚度，值越大越硬
Damping阻尼，值越大衰减越快
Gravity重力影响
Wind风力方向和强度

🎯 弹簧控制器

为对象添加弹簧控制器，实现弹性跟随效果。

使用场景：

  • 次级运动（Secondary Motion）
  • 惯性跟随效果
  • 弹性动画

使用步骤：

  1. 选择要添加弹簧效果的对象
  2. 指定跟随目标
  3. 调整弹簧参数
  4. 测试效果

📦 BoxMan 方块人

快速创建方块练习人模型，用于动画练习和测试。

功能：

  • 一键创建带绑定的方块人
  • 预设多种体型
  • 包含基础控制器

使用步骤：

  1. 打开 BoxMan 工具
  2. 选择体型预设
  3. 点击创建
  4. 开始动画练习

用途

非常适合动画初学者进行基础练习，无需复杂的角色模型
