# 快速入门

5 分钟快速上手 BsKeyTools。

---

## 🎛️ 界面概览

![界面预览](https://raw.githubusercontent.com/AniBullet/BsKeyTools/main/BsKeyToolsPrev.gif)

工具界面分为三个主要区域：

| 区域 | 说明 |
|:-----|:-----|
| **顶部工具栏** | 关键帧选择、移动、帧范围控制 |
| **快捷按钮区** | 显示控制、常用功能一键访问 |
| **扩展面板** | 4 页签 × 12 按钮，可自定义排序 |

---

## ⚡ 常用功能速查

### 关键帧操作

| 功能 | 说明 |
|:-----|:-----|
| 选择帧前/后 | 快速选择当前帧前/后的关键帧 |
| 移动帧 | 批量移动选中的关键帧 |
| 选定帧范围 | 将预览范围设为选中帧首尾 |
| 首尾帧范围 | 将预览范围设为全部帧范围 |

### 显示控制

| 功能 | 说明 |
|:-----|:-----|
| 秀剪影 | 切换剪影显示模式 |
| 冻模型 | 一键冻结/解冻模型 |
| 隐 Bone | 显示/隐藏 Bone 骨骼 |
| 隐 Biped | 显示/隐藏 Biped 骨骼 |
| 洋葱皮 | 运动重影显示 |
| 跳帧播 | Blocking 动画预览 |

### 快捷工具

| 功能 | 说明 |
|:-----|:-----|
| 快备份 | 快速备份当前场景 |
| 时光机 | 快速打开历史文件 |
| 暴力贴 | 动画复制粘贴 |
| 动画库 | Pose 库 / 参考库 |
| 层管理 | 图层管理器 |
| 选择集 | 选择集工具 |

---

## 🔧 扩展面板

点击主面板底部的页签按钮，可切换 4 个扩展面板（共 48 个工具按钮）。

::: tip 自定义排序
右键扩展面板的按钮可以调整按钮顺序
:::
